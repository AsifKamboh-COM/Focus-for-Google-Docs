# Chrome Storage
HashMap of `tabId-${number}` and `settings`. Examples:
    * `tabId-936794675`: 'on' | 'off'
    * `tabId-936794680`
    * `settings`: {
        fullScreen: boolean;
        autoOn: boolean;
        printLayout: boolean;
    }
